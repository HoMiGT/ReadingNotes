# 2 应急救护

实施急救的目的是“保存生命，减轻痛苦，防止病情或伤势恶化，促进复原”    
急救不仅限于进行**人工呼吸**、**包扎伤口**或将伤者**送往医院**，握住一个人的手、安慰受惊吓的人、以及奉献自己的一片爱心也是。    
应急救护的特点：
- 现场的
- 群众性的
- 初步的
- 自愿无偿的

应急救护的原则：
- 保证安全
- 挽救生命
- 防止恶化
- 心理支持
- 协同救护

应急救护人员：当疾病或意外伤害发生后，能够立即为伤病员实施应急救护的人。    
红十字救护员：参与培训，通过考试，获取证书即可成为红十字救护员。要掌握应急救护的基本技能，有爱心和社会责任感，当意外伤害或急症发生时，能够立即为伤病员实施应急救护。    

红十字救护员的基本任务：
- 确认现场安全
- 正确判断伤情
- 迅速呼救求援
- 实施紧急救护

民法典规定，主动救助致损害人伤害的，不承担民事责任

应急救护程序：
- D 评估环境(Danger)
- R 检查反应(Response)
- A 检查气道(Airway)
- B 检查呼吸(Breathing)
- C 检查循环(Circulation)
- D 检查清醒程度(Disability)
- E 暴露检查伤情(Exposure)

确保自身安全，做好自身防护：处理伤病员伤口、清洗衣物、处理血迹或秽物、处理自己的伤口、减少口对口的接触。    
防护用品：手套、口罩。必要时也可用塑料袋替代手套接受污染的物品。    

自身能力与现场资源的评估：
- 安全救人
- 科学救人
- 智慧救人
- 协同救人

成人的正常生命体征：
- 意识清醒
- 呼吸12-20次/分钟
- 心率60-100次/分钟
- 瞳孔等大等圆，对光反射良好

检查伤情：按头、颈、胸、腹、背、下肢、上肢顺序检查，尽可能在2-3分钟内完成    
识别伤情：
- 从头到脚
- 有意识吗
- 有呼吸、心跳吗
- 口、鼻有异物阻塞吗
- 有明显大出血吗
- 有骨折吗
- 有奇怪的味吗
- 有烧伤或其它吗

意识障碍分级
- I级 **嗜睡**：能唤醒，能用言语做出反应
- Ⅱ级 **昏睡**：较强刺激，能唤醒
- Ⅲ级 **浅昏睡**：对疼痛刺激有反应
- Ⅳ级 **深昏睡**：对疼痛刺激无反应

学会呼救    
先自报姓名再说清以下信息：
- 事发地点(详细定位)
- 事故类型、原因
- 受伤人数与伤情(群发伤)
- 请对方先放电话
- 最好用另一部电话拨打亲朋(有限考虑强壮男性)

检伤分类：
- 使用条件：当大规模伤害事件发生时，现有急救资源与伤员数量不对称(资源缺乏)时使用
- 使用目的：是以有限资源抢救更多有望存活的伤员
- 急危重伤病员优先
- 同等伤情，妇女、儿童、老人优先

检伤不是目的，当检伤与抢救发生冲突时，应以救命为先    
检伤重应该特别主义那些“不声不响”、反应迟钝的伤员，避免检查遗漏    

针对溺水者无须控水，而是立即心肺复苏并注意保暖

待援时不能做什么：
- 不要做没把握、不会做的事
- 不贸然进入可能带电/有毒等危险区域施救
- 不乱翻或随意拖拽摇晃伤员
- 不拔除刺入体内的异物
- 不还纳突出体外的组织
- 不喂食重创或意识不清的伤员大量饮水或食物
- 不丢弃离体的残肢或组织
- 不堵塞头部外伤者耳鼻流出的血水
- 不乱服用或乱擦药物
- 不搬开压在身上超过1小时的重物


救护员在急救车到达之前：
- 能够 自我保护
- 能够 识别伤情
- 能够 正确呼救
- 能够 正确施救
- 了解 哪些不能做
- 懂得 心理救援


# 3 心肺复苏(CPR)
心肺复苏(CPR)是一种紧急救护技术，用于在心脏骤停或呼吸停止的情况下恢复心脏和呼吸功能。它旨在通过人工提供心脏按压和人工呼吸来维持血液循环和氧气供应，以保护受害者的大脑和其他重要器官。

识别心脏骤停：
- 无反应(无意识)
- 无呼吸或仅有喘息样呼吸
- 无脉搏(由专业人员检查)

脑细胞耐氧极限
无论什么原因造成的心脏骤停，常温下，脑细胞完全缺氧3-4分钟时缺氧耐受的极限，8分钟后将出现不可逆转。高温下，脑细胞完全缺氧超过2-3分钟，即发生坏死。   

**黄金4分钟**，没延长抢救时间1分钟，成活率下降10%

CPR的关键步骤：
- C 胸外心脏按压
- A 开放气道
- B 人工呼吸

CPR的操作流程
- 观察环境
- 判断意识
   - 成人、儿童： 拍打双肩，轻拍重喊
   - 婴儿：拍打足底，观察反应
- 翻转体位(仰卧位)
   - 一手扶住头颈部
   - 一手扶住对侧腋下
   - 保证脊柱呈轴向整体翻转至复苏体位
- 判断呼吸
   - "看、听、感觉"约10秒
   - 用千位计数法(默念1001)
- 呼救求援
- 胸外按压
   - 成人按压要领：  **成人按压与人工呼吸比 30:2  垂直按压**
     - 位置：胸骨正中下半部
     - 深度：5-6cm
     - 频率：100-120次/分钟
     - 保证每次按压后胸廓完全回弹
     - 尽可能减少按压中断(中断不超过10秒)
   - 儿童(1岁-青春期)
     - 单手或双手 **双人操作时按压与呼吸比为15:2, 单人操作30:2**
     - 深度：5cm左右
   - 婴儿(不足1岁，新生儿除外)
     - 俩手指或环抱法(2人以上) **双人操作时按压与呼吸比为15:2，单人操作30:2**
     - 深度：4cm左右
- 尽早AED除颤
- 清理口腔异物
- 打开气道(仰头举颌法-颈椎未损伤)
- 人工呼吸
  - 成人: (90度)
     - 吹气同时眼睛看胸廓有无起伏，胸廓抬起即可
     - 吹气量：500-600ml(平静呼吸)
     - 吹气时间>1秒
     - 不得过度通气
   - 婴儿：(30度)
     - 满月至1岁以内孩子：口对口鼻
     - 缓慢(超过1秒)吹气，胸廓抬起即可
     - 频率: 12-20次/分


**五循环为一周期，每周期检查患者情况，不超过10秒**

中断救援的特征：
- 活了：患者被救活
- 来了：专业救护人员到场
- 有危险：抢救有致命危险需要紧急转移

胸外按压禁忌症
- 胸廓外伤
- 胸廓畸形
- 肋骨骨折
- 心包填塞

注意事项：
- 心肺复苏流程成人顺序按照 C-A-B
- 溺水患者，按照A-B-C流程操作
- 现场救护争分夺秒
- 不要随意搬动伤病员，主义保护脊柱
- 要在坚硬的平面伤进行心肺复苏操作
- 严禁在有呼吸和心跳的人身上进行CPR操作训练

高质量的心肺复苏要求：
- 定位准确，按频率100-120次/分钟，深度5-6cm，儿童5cm，婴儿4cm
- 每次按压后让胸廓完全回弹
- 持续按压，中断时间小于10秒
- 吹气避免过度通气
- 尽可能快地使用AED
